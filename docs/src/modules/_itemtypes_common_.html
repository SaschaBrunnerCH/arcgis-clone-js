Protected Const PLACEHOLDER_SERVER_NAME

PLACEHOLDER_SERVER_NAME: string = "{{organization.portalBaseUrl}}"
  • Defined in itemTypes/common.ts:32

A parameterized server name to replace the organization URL in a Web Mapping Application's URL to itself; name has to be acceptable to AGOL, otherwise it discards the URL, so substitution must be made before attempting to create the item.
